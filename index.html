常忘記的設定

🔑
API Key 放在哪？怎麼用？ 常忘
Anthropic（Claude）
1console.anthropic.com/settings/keys → Create Key
2Key 只顯示一次，立刻複製存好
3Cloudflare Worker：Settings → Variables → 加 ANTHROPIC_API_KEY
4本地：export ANTHROPIC_API_KEY=sk-ant-...
OpenAI（ChatGPT）
1platform.openai.com/api-keys → Create new secret key
2注意 Project vs Organization key 差異
☁️
Cloudflare Worker 設定流程 常忘
1dash.cloudflare.com → Workers & Pages → Create
2Edit Code → 貼上 proxy script → Deploy
3Settings → Variables → 加 ANTHROPIC_API_KEY（Encrypt ✓）
4Worker：fda510k-proxy（帳號：gogoyankee）
5CORS 記得加 Access-Control-Allow-Origin: *
6改完要重新 Deploy 才生效
🚀
GitHub Pages 部署流程 常忘
1Repo → Settings → Pages → Source：Deploy from a branch
2Branch main，資料夾 / (root) → Save
3網址：chiuchangru.github.io/[repo]/
4Push 後約 1–2 分鐘生效，Actions tab 看進度
常用 Repo
  • ChiuChangRu/litdb
  • ChiuChangRu/510K
  • ChiuChangRu/center（首頁）

SHA-256 密碼保護怎麼加 常忘
1產生 hash：crypto.subtle.digest('SHA-256', encoder.encode(pwd))
2把目標 hash 硬編碼在頁面（不是明文密碼）
3登入後 token 存 sessionStorage，重整自動清除
4510K Explorer 已實作此模式可直接參考
📦
GitHub Contents API 推檔 常忘
1先 GET 取得 SHA：GET /repos/:owner/:repo/contents/:path
2PUT 帶入 sha、message、content（base64）
3Token 需要 repo scope，存在 Cloudflare Worker 環境變數
4漏掉 sha → 409；content 忘記 btoa → 422
